# 番茄钟 PWA 设计文档

**日期**: 2026-07-01

## 概述

开发一个极简番茄钟应用，仅 Android，技术方案为 PWA（Progressive Web App）。单 HTML 文件包含所有界面和逻辑，部署到 GitHub Pages。

## 技术选型

- **方案**: PWA（渐进式 Web 应用）
- **语言**: HTML + CSS + JS（原生，无框架）
- **部署**: GitHub Pages
- **存储**: localStorage
- **组成文件**: index.html, manifest.json, sw.js, bell.mp3

## 核心功能

1. 倒计时器（默认 25 分钟工作 / 5 分钟休息）
2. 支持自定义工作时长和休息时长
3. 开始/暂停/重置按钮
4. 工作阶段和休息阶段自动切换
5. 时间到播放铃声 + 浏览器通知 + 振动
6. 设置面板（可展开/收起）
7. 通过 localStorage 记住用户设置

## 页面布局

- 顶部: 番茄 emoji 图标
- 中间: 圆形倒计时显示（SVG 环形进度条）
- 下方: 开始/重置按钮
- 状态标签: 工作中/休息中
- 底部: 设置按钮（点击展开时长调节）

## 配色方案

- 工作阶段: 番茄红 #E53935，对应圆形进度条颜色
- 休息阶段: 绿色 #43A047，对应圆形进度条颜色
- 背景: 浅灰白 #F5F5F5
- 文字: #333

## PWA 配置

- manifest.json: 应用名称、图标、全屏模式、主题色
- sw.js: 缓存关键文件，支持离线访问
- 支持"添加到主屏幕"后全屏运行

## 非功能需求

- 页面加载时间 < 2 秒
- 离线可运行（已缓存的资源）
- 支持 Chrome for Android 和 Samsung Internet
- 响应式设计，适配主流手机屏幕
